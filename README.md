# Hallein App
## Setup
### Yarn
Install [Node.js](https://nodejs.org/en/download/) and [Yarn](https://yarnpkg.com/en/docs/install)

### Cordova
Open commandline in halleinApp/
```
$ cordova prepare
```
Then type
```
$ npm install -g gulp-cli
$ yarn
```
At last compile your files to www/
```
$ gulp
```
